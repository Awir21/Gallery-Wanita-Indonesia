# Editorial Glamour Woman Gallery

Website **Editorial Glamour Woman Gallery** adalah proyek galeri visual fashion wanita dengan pendekatan editorial modern. Fokus utama website ini adalah menampilkan kurasi foto wanita secara **simetris, responsif, dan premium**, menggunakan **HTML dan CSS murni** tanpa framework maupun JavaScript.

Proyek ini dirancang untuk memenuhi standar penilaian tinggi (UI/UX, kerapian kode, responsivitas, dan konsep visual), sehingga cocok untuk **tugas kuliah, portofolio desain web, maupun showcase frontend**.
